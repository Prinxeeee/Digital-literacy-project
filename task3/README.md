# Task 3 – Coding & Collaboration Platforms

This folder showcases evidence of my active engagement with coding and collaboration tools, highlighting both my technical development and my ability to create and share digital content effectively.

Part A – Coding Platform
For this section, I used HackerRank to practice and strengthen my programming skills by solving a beginner-level problem. This activity allowed me to apply logical thinking, understand basic coding concepts, and improve my problem-solving abilities. It also introduced me to the structure of coding challenges and helped me become more confident in writing and testing code within an online platform.

Part B – Google Forms
In this part, I designed a Digital Literacy Awareness Quiz using Google Forms. The purpose of this quiz was to assess and reinforce understanding of key digital literacy concepts, such as online safety, digital tools, and responsible communication. The folder includes screenshots of both the form interface and the automatically generated response sheet, demonstrating how data can be collected and analyzed efficiently.

The link to the Google Form has been included in the main README file, allowing easy access for others to view and attempt the quiz. Overall, this task enhanced my skills in digital collaboration, content creation, and data organization.
